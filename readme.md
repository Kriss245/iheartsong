## IHEARTSONG - WEBSITE MANAJEMEN PLAYLIST MUSIK

IHEARTSONG merupakan sistem yang dirancang untuk mengelola playlist musik untuk pengalaman mendengar yang personal dan terorganisir seperti: mengatur, menambahkan, dan nikmati koleksi musik favorit Anda dengan mudah.

Beberapa fitur diantaranya: 

- Daftar / Register
- Masuk / Login
- Tambah musik
- Tambah musik ke playlist yang ada
- Filter berdasarkan playlist
- Pencarian musik dan playlist
- Penyortiran sesuai abjad dari a-z ataupun sebaliknya
- Tampilkan 10/25/50/100 data per halaman.
- Tambah playlist

Dalam perancangan website ini, kami menggunakan PHP versi 7.4 dan Framework Laravel 8.

## Cara Install dan Menjalankan

`$ >git clone https://github.com/Kriss245/iheartsong.git`

`$ > cd iheartsong`

`$ > composer install or composer update`

`$ > cp env.example .env`

`$ > Set up .env file`

`$ > php artisan key:generate`

`$ > php artisan migrate`

`$ > php artisan serve`

Kemudian, Silahkan buka browser dan ketikkan: http://localhost:8000
